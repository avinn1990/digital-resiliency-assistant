# Enterprise Security Governance (evaluation service pack)

This evaluation service pack defines the **capabilities** and **reference questions** used by the LLM interview to assess **Enterprise Security Governance**.

- **service_id**: `enterprise-security-governance`
- **Capability id prefix**: `esg-` (e.g. `esg-00`)
- **Question id format**: `rq-esg-00-1`

## Validate content

Run:

```bash
python3 "evaluation-services/Enterprise Security Governance/validate_evaluation_content.py"
```

Expected output:

```text
ok: N capabilities, M questions, 1:1 capability mapping enforced
```
